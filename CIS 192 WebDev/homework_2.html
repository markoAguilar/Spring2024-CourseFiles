HW#2 Intro to HTML/CSS/Javascript

CSS stands for Cascading Style Sheets! CSS rules specify specify styling properties for specific HTML elements.
CSS rules are placed withing <style> < /style> tags. Theses rules are placed within the head element of html in the
beginning. In the style tags are style element in the format of
element{
property:value;this-property:value;
that-property:value;
}
Each element in the <style> tag is named after the elements in the body that will be formatted according to the properties and
values. There is no need for < or > signs just the element name.

Javascript is a programming language that runs in the browser and allows for page interactivity. It can be included in either the <head>
or its <body> parts
