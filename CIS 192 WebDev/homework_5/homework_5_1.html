<script> & <style> Tags

<script>: Allows executable code that the browser assumes to be JavaScript unless indicated otherwise. Common attributes of the <script> container are:

  • type attribute: optional, indicates content type if not JavsScript
  • src attribute: provides the url of an external file containing JavaScript code
  • To include the file with external JavaScript simply use the <script> tag with the src attribute assigned to the file location along with the closing <script> tag. The file should end with the (.js) extension and can be either a file at a URL or a local file.

<style>: allows presentation directive usually in CSS. A <style> tag is placed before the body of an HTML document

Unlike other HTML tags, the content within the opening and closing <style> and <script> tags is not displayed to the user.
