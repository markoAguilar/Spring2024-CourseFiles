Part 2 : Basic Selectors

Element, Class, & ID Selectors

Three common selector types are:

  • The element selector: matches elements with the specified element names
    Ex: p {color: blue;} selects all p elements
  • The class selector: with the format .className{ }, matches elements with the matching class attribute
    Ex: .notice {color: blue;} selects all elements with the class="notice advisory" attribute.
    • Notice how the class attribute can be assigned more than one class. Furthermore, while id attributes must be unique&44; many elements can use the same HTML class name,
  • The ID selector: with the format #idName { } matches elements with the matching id attribute
    Ex. #byLine {color: blue;} selects elements with the id="byLine" attribute

Descendant Selector

The descendant selector with the format firstSelector secondSelector { } matches secondSelector elements contained within firstSelector elements

Pseudo‐class Selector

Pseudo‐selector: with the format :pseudoClassName { } matches elements based on user behavior or element metainformation

Example pseudo‐class selectors:

  • :disabled: Element is disabled
  • :hover: Mouse is hovering over element
  • :empty: Element has no child elements
  • :lang: (language) Element contains text in a specific language
  • :nth‐child: Element is the parent elements nth‐child

class selectors and pseudo‐class selectors can combine with element names.
Ex. a:hover selects only the anchor tag the mouse is hovering over
Ex. span.highlight selects only span elements that have the class attribute value highlight
