First Heading

Comments are enclosed by less than sign!-- -->
the p tag is a paragraph element in the body
the br tag will create a new line. They should not be used in headings
and the hr tag will create a horizontal line

Note that these tags do not need to be closed. Closed tags are reserved for elements for which their
modifications apply to the entire enclosed content.


Heading tags may also come in the format of hn where n is a number
generally 1-6. The size of the text is set to the number. Below is
h4 followed by h3 and h2. The size decreases with higher numbers.

The ul tag is an unordered list with list items enclosed by the li tag

    This unordered list
  • First item
  • Second Item

The unordered list tag contains an attribute style that takes a string value "list-style-type: type;"
The type of the bullet point is then set to the value of type such as circle or square

The ol tag is an ordered list which numbers the li elemements in your list
It also contains a type attribute that takes in a string argument of either type = "a, A, i, I, or 1"
Furthermore you can also have lists within lists, for example an unordered list with square bullets
inside of an order roman numbered list

  1. My first step
  2. My second step
  3. My third Step
    1. Sub Step A
    2. Sub Step B
    3. Sub Step C

The anchor tag or a tag contains an href attribute that takes in the argument pointing to another page or file
There are absolute and relative file types. Absolute is the url to a webpage, and relative can be like a path
or local location of another html file. The enclosed content will be what is displayed as the clickable text to
the absolute or relative file.

The image tag has attributes src="relative filepath" width="###" height="###" alt="name for the image"
The image tag does not need to be closed as it does not need to modify any enclosing data
