Additional Form Widgets

Date Picker:

  • Date picker is an input element set by the input attribute "date"
  • Two common date attributes are max(furthest date permitted) and min(earliest date permitted). values for these attributes must be in the form "yyyy-mm-dd"

Color Picker:

  • Color picker is an input element set by the input attribute "color"

Number Input:

  • Number Input is an input element set by the input attribute "number"
  • Two common number attributes are max(max number) and min(min number)

Combo Box:

  • A combo box is the combination of text box and drop-down menu all in one widget.
  • An <input> element, with a list attribute and list attribute value of "idOfDatalistContainer"
  • A <datalist> container with included <option> elements

Special Input Text

  • Four <input> element types exist for entering special text:
    • type="url" for typing in a URL:
    • type="tel" for typing in a phone number
    • type="email" for typing in an email
    • type="search" for typing in search terms

It is important to specify what input element to use for each type of text being requested. Some browsers display different keyboard layouts for users

other input attributes include:

  • pattern: also known as a regular expression. Specifies the pattern that the users input must follow. Learn more about Regular Expressions here
  • required: no value assigned, the field cannot be left empty if required is present
  • step: sets the amount by which the value changes. for a range input element for example
